Welcom to Dog Identification Service!

What is this website?

Dog Identification Service is an example project that uses django and trained pytorch model to allo0w the user to identify the dog in an image. The image can be uploaded by the user and the model will identify the dog in the image.

How to use this web service?

The detailed instructions can be found in About subpage

About the model

The model used for the app has been trained on the Stanford Dog Dataset which is a collection of 20580 images of different dogs of 120 breeds.

The model bases on residual network ResNet152 architecture.

The model was trained PyTorch package for Python using transfer learning approach basing on the resnet model trained on image net while training layer3 and layer 4 of the networks with different learning rates.
